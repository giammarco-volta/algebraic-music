# Algebraic Music Representation

This repository contains a minimal C++ implementation of the system described in the paper:

**"An Algebraic Representation of Notes and Intervals for Algorithmic Composition"**

## Overview

The system represents notes and intervals as elements of a cyclic integer domain, preserving:

- enharmonic distinctions (C♯ ≠ D♭)
- interval structure
- efficient computation via integer arithmetic

- `algebraic_music.hpp` — core implementation
